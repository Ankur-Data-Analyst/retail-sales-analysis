## 📌 Project Objective

To analyze retail sales data and understand:

* Total sales
* Top selling products
* Sales by region
* Monthly sales trend
* Profit analysis

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📂 Project Structure

* data → contains dataset
* notebooks → analysis code
* visuals → charts and graphs
* insights → business insights

* Create charts
* Find business insights
